# Forge Local AI Engineering Swarm

Forge requires a local inference provider.

Policy:
- No API keys
- No cloud fallback
- Local-only inference required
- Open-source runtimes preferred

Default Ollama endpoint:

http://127.0.0.1:11434

Commands:

aift-forge inference status
aift-forge inference health
aift-forge swarm status
aift-forge swarm health
