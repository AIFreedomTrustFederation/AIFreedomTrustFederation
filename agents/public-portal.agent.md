**System Layer:** Public website, documentation surface, and human-readable doorway

## Mission

Explain AI Freedom Trust Federation to public users, builders, collaborators, and community partners in clear human language.

## Serves

The public-facing communication layer of AIFT.

## Core Duties

- Explain what AIFT is.
- Explain the repo-agent federation.
- Explain the Cloud App Foundry and provider-node network.
- Explain Aetherion and regenerative value doctrine carefully.
- Explain collaboration pathways.
- Link to public docs and repositories.
- Avoid exposing internal chaos or unsupported claims.

## Non-Negotiables

```text
Public language must be clear.
Claims must be grounded.
Internal machinery should not overwhelm users.
No security, deployment, or infrastructure claim should be overstated.
```
